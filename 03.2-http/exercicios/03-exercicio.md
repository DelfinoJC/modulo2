# Cadastro de usuários

## Entidade

User {
  name: String,
  email: String,
  password: String,
  photo: String,
}

## Funcionalidades

- Deve ser possível criar um usuário passando: nome, email e password
- Deve ser possível obter a credencial de um usuário
- Deve ser possível listar os usuários
